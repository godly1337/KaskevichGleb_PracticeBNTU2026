Анализ инерциальных данных (IMU) мобильного робота


ОПИСАНИЕ

Практическая работа по обработке и визуализации данных с акселерометра
и гироскопа мобильного робота. Данные берутся из датасета MAGF-ID.

Репозиторий датасета: https://github.com/ansfl/MAGF-ID
Статья с описанием данных: https://link.springer.com/article/10.1038/s41597-024-03917-6


ЧТО ДЕЛАЕТ СКРИПТ

1. Загружает CSV-файл с показаниями IMU (акселерометр + гироскоп).
2. Строит графики ускорений по осям X, Y, Z.
3. Строит графики угловых скоростей по осям X, Y, Z.
4. Автоматически определяет и отмечает на графиках моменты старта,
   остановки и резких поворотов.
5. Выводит в консоль сводку по найденным событиям.

Если указанный CSV не найден, скрипт генерирует тестовые данные, чтобы
можно было проверить его работу без реального датасета.


УСТАНОВКА

Перед первым запуском нужно установить библиотеки:

pip install pandas numpy matplotlib scipy


ИСПОЛЬЗОВАНИЕ

1. Скачать CSV-файл из датасета MAGF-ID.
2. Указать путь к файлу в переменной CSV_PATH в начале code.py,
   либо передать его аргументом при запуске:

   python code.py путь/к/файлу.csv

3. Если автоматическое определение столбцов не сработало (названия
   отличаются от стандартных), прописать их вручную в COLUMN_OVERRIDE
   в начале файла.


РЕЗУЛЬТАТ

После запуска в папке output_plots появятся:

accelerometer.png  — ускорения по X, Y, Z и модуль ускорения
gyroscope.png      — угловые скорости по X, Y, Z и их модуль
overview.png       — сводный график с отмеченными событиями

В консоли выводится время каждого обнаруженного старта, остановки
и поворота.
